Create a submission

  • Within your docker container input data is located at /data.csv
  • This CSV file has the columns: original_filename;path;terms;origin
  • Files can be read from directory /input/. Prefix any path given in column path by /input/.
  • Any output files must be placed in directory /output/.
  • Ask your challenge conductor which output files are required in order to correctly evaluate your submission.
  • Make sure your container is not using any of /data.csv, /input/, /output/ as they will be used to mount volumes.
  • Push your docker image to the registry provided by your challenge conductor or they can not be used for evaluation.
